    Logging

    The debug library can provide logging during development. Two different logging namespaces are provided, node-fhir- client:info logs each request and response, and node-fhir-client:error logs errors. To enable logging during development, add one of the namespaces to the DEBUG environment variable, or use node-fhir-client:* to enable both.

    $ DEBUG=node-fhir-client:* node examples/index.js
